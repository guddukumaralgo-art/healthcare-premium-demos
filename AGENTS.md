# Healthcare Premium Demo Website Rules

Build modern, premium, animated healthcare demo websites.

Tech stack:
- React + Vite
- Tailwind CSS
- Framer Motion / Motion
- Lucide React icons
- CSV or JSON data

Design style:
- Premium Framer-style landing pages
- Clean healthcare look
- White background
- Soft blue, teal, green, lavender gradients
- Rounded 2xl cards
- Soft shadows
- Large hero headline
- Beautiful spacing
- Mobile-first responsive layout
- Smooth professional animations

Animation rules:
- Use fade-up on scroll
- Use subtle floating cards
- Use animated counters only if useful
- Avoid too many flashy animations
- Website must feel trustworthy, not childish

Healthcare rules:
- Do not make fake medical claims
- Do not add fake awards
- Do not add fake doctor qualifications
- Do not use fake patient reviews unless clearly marked as demo
- Always add disclaimer:
  "Unofficial sample concept — created for design preview only."

Batch rules:
- Do not delete old batches
- Do not overwrite existing generated websites unless asked
- Generate only requested batch
- Keep deployment output inside publish/
